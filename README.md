# ABC Fitness Studio Website

This project is a multi-page website for ABC Fitness Studio, built using HTML and CSS. The goal of the site is to present class offerings, schedules, and basic studio information in a clean, easy-to-navigate layout.

## Pages Included

- Home (index.html) – Overview of the studio, services, and hours

- Classes (classes.html) – Weekly class schedule and class descriptions

- Gallery (gallery.html) – Images and fitness gear

- About / Contact (about-contact.html) – Studio information and contact form

- Thank You (thank-you.html) – Confirmation page after form submission

## Technologies Used

- HTML5

- CSS3

## Styling Approach

The site uses modular CSS files to keep styles organized and avoid repetition. Shared styles are reused across pages, while page-specific styles are applied only where needed.

## How to View the Site

Download or open the project folder

Open index.html in a web browser

Use the navigation menu to explore each page
